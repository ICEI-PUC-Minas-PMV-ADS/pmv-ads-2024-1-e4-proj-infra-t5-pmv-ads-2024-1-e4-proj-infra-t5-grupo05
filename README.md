# My Closet Online


![myclosetapp](https://github.com/ICEI-PUC-Minas-PMV-ADS/pmv-ads-2023-2-e3-proj-mov-t4-2023-e3-projmovt4-time2-myclosetweb-atualizado/assets/104511336/01dd6bbd-66b9-414b-84f1-6ca79bf65f83)

`Tecnologia em Análise e Desenvolvimento de Sistemas`

`Projeto: Desenvolvimento de uma Aplicação Móvel em um Ambiente de Negócio`

`Eixo 4`

Este projeto trata-se do desenvolvimento de uma aplicação funcional, com o intuito de dar uma plataforma ao cliente onde ele possa visualizar suas peças de roupas de forma mais organizada e eficiente.
